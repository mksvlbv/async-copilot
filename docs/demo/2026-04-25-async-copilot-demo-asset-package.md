# Async Copilot Demo Asset Package

This is the reviewer-facing demo bundle for the current Priority 5 hiring surface.

Use it when a reviewer needs the fastest path from project context to visual proof.

Use this as the first visual preview in the README and repository page. It shows the product promise and the core triage workspace without requiring a reviewer to run the app first.

- `docs/demo/2026-04-20-async-copilot-reviewer-walkthrough.md`

## Reviewer sequence

1. Open the README and watch `public/demo/demo.gif`.
2. Scan `Reviewer Quick Read` and `Trust Evidence Snapshot`.
3. Open the walkthrough if a narrated 60-90 second demo is needed.
4. Use the screenshot gallery as static backup evidence for the main surfaces.

## Current boundary

This package is a hiring-surface artifact only. It does not expand product scope, runtime behavior, export formats, Gmail sync, Slack behavior, analytics, or evaluation breadth.
